## Installation


### Install packages
```bash
$ npm install
```

### Generate prisma types
```bash
$ npx prisma generate
```

### Deploy prisma schemas
```bash
$ npx prisma migrate deploy
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
